マニュアル

          
このアプリは、学校の iPad で物理実験の測定データを表入力し、そのまま webR で解析するための単一 HTML ツールです。サーバーへデータ送信せず、GitHub Pages 上でも静的配信で動きます。


          

            

              
基本の流れ

              
    
                
  1. 左側の表に測定値を入力します。
    2. 
                
  2. 1 行目は列名です。日本語や記号を入れても R 互換名へ自動調整されます。
    3. 
                
  3. 右上のエディタに R コードを書いて「コード実行」を押します。
    4. 
                
  4. 結果は右下の「コンソール」または「グラフ」で確認します。
    5. 
              

            

            

              
シート操作

              
    
                
  • 下部タブでシート切り替え
    • 
                
  • ＋ボタンで新規シート追加
    • 
                
  • 「シート名変更」でタブ名変更
    • 
                
  • 「シート削除」で不要シート削除
    • 
              

            

          


          
get.input() の使い方

          

            
              

                書き方
                意味
              

            
            
              

                df <- get.input()
                現在のアクティブシートを data.frame として取得します。
              

              

                df <- get.input("Sheet2")
                シート名を指定して取得します。
              

              

                df <- get.input(2)
                1 始まりのシート番号で取得します。
              

            
          


          
JSON 保存と復元

          
「設定保存 (JSON)」は、単なる表データだけではなく、できるだけ作業中の状態も保存します。たとえば次の情報を含みます。

          
    
            
  • すべてのシート内容とアクティブシート
    • 
            
  • 右上のスクリプトエディタ内容
    • 
            
  • 現在表示中のページ（ワークスペース / マニュアル）
    • 
            
  • コンソール / グラフの選択状態
    • 
            
  • コンソール出力履歴と REPL 履歴
    • 
          

          
ただし、R ランタイム内部のオブジェクトそのものは保存されません。JSON を読み込んだ後は、必要に応じてスクリプトを再実行してください。


          
コード例

          
df <- get.input()
summary(df)

plot(
  df$time_s,
  df$distance_m,
  pch = 19,
  xlab = "time_s",
  ylab = "distance_m"
)


          
補足

          
    
            
  • Ctrl / Command + Enter でスクリプト実行できます。
    • 
            
  • コンソール入力欄では上下キーで実行履歴をたどれます。
    • 
            
  • ggplot2 を使うときは初回のみ読み込みが少し長くなることがあります。
    • 
            
  • 画面サイズを大きく変えた後は、必要に応じてコードを再実行するとグラフが最新サイズに更新されます。
