マニュアル

          
このアプリは、学校の iPad で物理実験の測定データを表入力し、そのまま webR で解析するための単一 HTML ツールです。サーバーへデータ送信せず、GitHub Pages 上でも静的配信で動きます。


          

            

              
基本の流れ

              
    
                
  1. 左側の表に測定値を入力します。
    2. 
                
  2. 1 行目は列名です。日本語や記号を入れても R 互換名へ自動調整されます。
    3. 
                
  3. 右上のエディタに R コードを書いて「コード実行」を押します。
    4. 
                
  4. 結果は右下の「コンソール」「グラフ」「データフレーム」で確認します。
    5.

  • 下部タブでシート切り替え
    •

    •

入力と出力

          
    
            
  • 「入力」では、Excel(xlsx)・CSV・TSV のデータ取込、設定(JSON)の読込、授業用サンプルの展開ができます。データ取込時は、取り込むシートや見出し行を確認できます。
    • 
            
  • 「出力」では、データ領域や最新のデータフレームを CSV / xlsx で書き出せます。R スクリプトのみ、設定(JSON)、一括(zip)の出力にも対応し、ファイル名には日時が付きます。
    • 
            
  • R スクリプトのみを出力すると、get.input() を含む行は自動でコメントアウトされます。詳細設定で CSV の文字コードやデータフレームのまとめ方を切り替えられます。
    • 
            
  • 設定(JSON)を読み込んだ後は、必要に応じてコードを再実行してください。
    • 
          


          
詳しい学習用ガイド

          
R の書き方、get.input()、よいグラフの作り方は、用途別の別ページにまとめています。

          
    
            
  • 生徒ガイド（データフレームから作図・欠損処理・誤差棒・回帰までを順に学ぶ）
    • 
            
  • スニペット集 / チートシート（コピペ用コードと関数早見表）
    • 
            
  • 教師ガイド（授業での使い方と R の背景）
    • 
            
  • 開発者向けリファレンス（内部構造・改変・再ホスティング）
    • 
          


          
補足の操作

          
    
            
  • Ctrl / Command + Enter でスクリプト実行できます。
    • 
            
  • コンソール入力欄では上下キーで実行履歴をたどれます。
    • 
            
  • スクリプト実行では head(df) や summary(df) などの結果もコンソールへ表示されます。
    • 
            
  • 右下の「データフレーム」タブでは、R セッション内の data.frame 一覧と先頭行プレビューを確認できます。
    • 
            
  • 左側シートは TSV をそのまま複数セルへ貼り付けできます。
    • 
            
  • ggplot2 を使うときは初回のみ読み込みが少し長くなることがあります。
    • 
            
  • 細かい作図調整は、R コード側で par() や set.paper_style() を使って設定してください。
    • 
            
  • 図の縦横比やサイズは set.figure(width, height) で指定できます（未指定なら描画エリアに自動フィット、reset.figure() で戻す）。
    • 
            
  • plot() を複数回呼ぶと図が複数になり、グラフタブ上部のサムネイルで切り替えられます。「全体表示」「ダウンロード」は選択中の図に対して働きます。
    •
